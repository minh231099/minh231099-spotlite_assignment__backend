## Directory structure
```
src
    ├───controllers -- Controller get Request and send Response
    ├───entities -- Database Modeling
    ├───interfaces -- Interfaces
    ├───repositories -- Data access 
    ├───routes -- Routing
    │   └───v1
    └───services -- Process logic before using parameters to retrieve data
```
